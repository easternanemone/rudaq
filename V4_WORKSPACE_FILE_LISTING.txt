HOW TO USE V4-DAQ WORKSPACE
================================================================================

Build v4-daq standalone:
  $ cd /Users/briansquires/code/rust-daq/v4-daq
  $ cargo build --features instrument_serial

Build with all features:
  $ cargo build --features full --release

Run tests:
  $ cargo test --lib --features instrument_serial

Check compilation:
  $ cargo check --features instrument_serial

Build hardware test example:
  $ cargo build --example v4_newport_hardware_test \
      --features instrument_serial --release

Output binary location:
  /Users/briansquires/code/rust-daq/v4-daq/target/release/examples/v4_newport_hardware_test

Deploy to hardware (maitai@100.117.5.12):
  $ rsync -avz target/release/examples/v4_newport_hardware_test \
      brian@100.117.5.12:/home/brian/bin/
